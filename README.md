# claw — Personal AI Workspace

This repository is a long-running personal AI workspace.
A resident AI assistant lives here, maintaining memory and continuity across sessions through files committed to this repo.

## Getting Started

Every new Copilot session should begin by reading [`AGENTS.md`](./AGENTS.md).
It defines who the assistant is, how it works, and where memory is stored.

| File / Directory | Purpose |
|---|---|
| [`AGENTS.md`](./AGENTS.md) | Core operating rules — read this first |
| [`MEMORY.md`](./MEMORY.md) | Long-term memory: facts, preferences, active projects |
| [`memory/daily.md`](./memory/daily.md) | Session / daily log — refreshed each day |
| `memory/<topic>.md` | Per-topic notes created as needed |
